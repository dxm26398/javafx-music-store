# JavaFX Music Store — MavTunes

A desktop application built with JavaFX and FXML for managing music and app purchases.

## Features
- Customer registration form with name, address, and contact info
- Music and app purchase selection with dynamic UI controls
- Dropdown menu for music genre selection (Pop, R&B, Jazz, Classical)
- Radio buttons for app type selection (Game, Productivity, Education)
- Checkbox logic that dynamically enables/disables relevant fields
- Data saved to output file using FileWriter

## Technologies
- Java
- JavaFX
- FXML
- MVC Controller pattern
